\section{Objetivo profundo}

El alumno debe comprender:

\begin{itemize}
\item cómo se implementa durabilidad real
\item cómo MVCC permite concurrencia sin locks
\item cómo LSM transforma la interacción con el disco
\end{itemize}

---

\section{Estructura sugerida (90 min)}

\subsection*{Bloque 1 (20 min): WAL}

\begin{itemize}
\item mostrar fallo sin WAL
\item mostrar recovery con WAL
\end{itemize}

\textbf{Ejercicio en clase:}
simular crash con log incompleto

---

\subsection*{Bloque 2 (25 min): MVCC}

\begin{itemize}
\item explicar versiones
\item hacer ejemplo con timestamps
\end{itemize}

\textbf{Momento clave:}
\begin{center}
“los reads no bloquean writes”
\end{center}

---

\subsection*{Bloque 3 (25 min): LSM}

\begin{itemize}
\item explicar secuencial vs aleatorio
\item dibujar flujo completo
\end{itemize}

---

\subsection*{Bloque 4 (20 min): integración}

\begin{itemize}
\item unir WAL + MVCC + LSM
\item discutir trade-offs
\end{itemize}

\begin{itemize}

\item ignorar costo de compaction
\item creer que MVCC no tiene costo
\end{itemize}
