\end{itemize}

\section{Preguntas sugeridas del jurado}
\begin{itemize}
\item ¿Qué restricción domina realmente el sistema?
\item ¿Qué pasa si hay partición de red?
\item ¿Qué anomalías aceptan?
\item ¿Cómo resuelven conflictos?
\item ¿Qué atacante modelaron?
\item ¿Cómo debuggearían una degradación en producción?
\item ¿Cuál es el mayor riesgo del diseño?
\end{itemize}

\section{Pregunta final}
¿Por qué este sistema está bien diseñado para el dominio elegido?

\begin{uakeybox}
La meta del jurado no es encontrar una arquitectura perfecta, sino verificar que el estudiante sabe defender un diseño coherente bajo restricciones explícitas.
\end{uakeybox}
